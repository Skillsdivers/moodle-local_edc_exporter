# Moodle Plugins Directory Submission Text

## Short Description

Export Moodle course completion data as European Digital Credential-compatible JSON credentials.

## Full Description

EDC Exporter is a local Moodle plugin that allows authorised users to generate, view, download, revoke and verify course completion credentials in a European Digital Credential-compatible JSON format. The plugin supports issuer configuration, visual credential covers, verification tokens, credential lifecycle management, Privacy API support and audit logging.
