Liberation Fonts
================

This folder contains TrueType font files used by local_edc_exporter to render generated credential cover images.

Bundled files:

- LiberationSans-Regular.ttf
- LiberationSerif-Regular.ttf
- LiberationSerif-Italic.ttf

License: SIL Open Font License 1.1.

The exact upstream version is not specified in the bundled font files. When updating these fonts, download a stable upstream release, replace the files in this directory, update ../thirdpartylibs.xml with the exact version, and run the plugin packaging checks again.
